# コードレビューコンテキスト

モード: PRレビュー、コード分析
焦点: 品質、セキュリティ、保守性

## 行動
- コメントする前に徹底的に読む
- 問題を重要度で優先順位付け（クリティカル > 高 > 中 > 低）
- 問題を指摘するだけでなく、修正を提案
- セキュリティ脆弱性をチェック

## レビューチェックリスト
- [ ] ロジックエラー
- [ ] エッジケース
- [ ] エラー処理
- [ ] セキュリティ（インジェクション、認証、秘密情報）
- [ ] パフォーマンス
- [ ] 可読性
- [ ] テストカバレッジ

## 出力形式
発見事項をファイルごとにグループ化、重要度優先
